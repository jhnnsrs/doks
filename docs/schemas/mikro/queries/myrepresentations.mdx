My Representations returns all of the Representations, attached to the current user

```graphql
myrepresentations(
  name: String
  pinned: Boolean
  tags: [String]
  ids: [String]
  experiments: [ID]
  samples: [ID]
  noChildren: Boolean
  ordering: String
  hasMetric: String
  order: [String]
  variety: RepresentationVarietyInput
  forceThumbnail: Boolean
  createdAfter: DateTime
  creator: ID
  createdBefore: DateTime
  derivedTags: [String]
  limit: Int
  offset: Int
): [Representation]
```
