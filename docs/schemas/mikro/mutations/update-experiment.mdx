 Update an Experiment
    
    This mutation updates an Experiment and returns the updated Experiment.

```graphql
updateExperiment(
  creator: String
  description: String
  id: ID!
  name: String!
  tags: [String]
): Experiment
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>updateExperiment.<b>creator</b></code>](#)<Bullet />[`String`](/docs/schemas/mikro/scalars/string) <Badge class="secondary" text="scalar"/>
> The user that created this experiment (defaults to the logined user)
> 

#### [<code style={{ fontWeight: 'normal' }}>updateExperiment.<b>description</b></code>](#)<Bullet />[`String`](/docs/schemas/mikro/scalars/string) <Badge class="secondary" text="scalar"/>

> 
>

### Type

#### [`Experiment`](/docs/schemas/mikro/objects/experiment) <Badge class="secondary" text="object"/>
> 
    An experiment is a collection of samples and their representations.
    It mimics the concept of an experiment in the lab and is the top level
    object in the data model.

    You can use the experiment to group samples and representations likewise
    to how you would group files into folders in a file system.
