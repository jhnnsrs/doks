Creates an Instrument
    
    This mutation creates an Instrument and returns the created Instrument.
    The serial number is required and the manufacturer is inferred from the serial number.
    

```graphql
createInstrument(
  detectors: [GenericScalar]
  dichroics: [GenericScalar]
  filters: [GenericScalar]
  lotNumber: String
  manufacturer: String
  model: String
  name: String!
  serialNumber: String
): Instrument
```
