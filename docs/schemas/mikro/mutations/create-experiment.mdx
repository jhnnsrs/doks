Create an Experiment
    
    This mutation creates an Experiment and returns the created Experiment.
    

```graphql
createExperiment(
  creator: String
  description: String
  name: String!
  tags: [String]
): Experiment
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>createExperiment.<b>creator</b></code>](#)<Bullet />[`String`](/docs/schemas/mikro/scalars/string) <Badge class="secondary" text="scalar"/>
> The user that created this experiment (defaults to the logined user)
> 

#### [<code style={{ fontWeight: 'normal' }}>createExperiment.<b>description</b></code>](#)<Bullet />[`String`](/docs/schemas/mikro/scalars/string) <Badge class="secondary" text="scalar"/>
> A short description of the experiment
> 

#### [<code style={{ fontWeight: 'normal' }}>createExperiment.<b>name</b></code>](#)<Bullet />[`String!`](/docs/schemas/mikro/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> A name for the experiment
> 

#### [<code style={{ fontWeight: 'normal' }}>createExperiment.<b>tags</b></code>](#)<Bullet />[`[String]`](/docs/schemas/mikro/scalars/string) <Badge class="secondary" text="list"/> <Badge class="secondary" text="scalar"/>
> Tags for the experiment
> 

### Type

#### [`Experiment`](/docs/schemas/mikro/objects/experiment) <Badge class="secondary" text="object"/>
> 
    An experiment is a collection of samples and their representations.
    It mimics the concept of an experiment in the lab and is the top level
    object in the data model.

    You can use the experiment to group samples and representations likewise
    to how you would group files into folders in a file system.
