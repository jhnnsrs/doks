Sharable Models are models that can be shared amongst users and groups. They representent the models of the DB

```graphql
enum SharableModels {
  GRUNNLAG_USERMETA
  GRUNNLAG_ANTIBODY
  GRUNNLAG_EXPERIMENT
  GRUNNLAG_EXPERIMENTALGROUP
  GRUNNLAG_ANIMAL
  GRUNNLAG_OMEROFILE
  GRUNNLAG_SAMPLE
  GRUNNLAG_REPRESENTATION
  GRUNNLAG_INSTRUMENT
  GRUNNLAG_OMERO
  GRUNNLAG_METRIC
  GRUNNLAG_THUMBNAIL
  GRUNNLAG_ROI
  GRUNNLAG_LABEL
  GRUNNLAG_FEATURE
  BORD_TABLE
}
```
