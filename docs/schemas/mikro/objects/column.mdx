A column in a table

A Column describes the associated name and metadata of a column in a table.
It gives access to the pandas and numpy dtypes of the column.

```graphql
type Column {
  name: String
  fieldName: String!
  pandasType: PandasDType
  numpyType: String
  metadata: GenericScalar
}
```
