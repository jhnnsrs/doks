Reservation

No description

type Reservation {
  id: ID!
  channel: String!
  happy: Boolean!
  viable: Boolean!
  allowAutoRequest: Boolean!
  node: Node!
  title: String
  template: Template
  provisions: [Provision!]!
  params: ReserveParams
  hash: String!
  status: ReservationStatus!
  statusmessage: String!
  callback: String
  progress: String
  createdAt: DateTime!
  updatedAt: DateTime!
  waiter: Waiter!
  app: LokApp
  creator: User
  reference: String!
  provision: Provision
  createdProvisions: [Provision!]!
  log(
  level: LogLevelInput
  createdAt: String
  o: String
): [ReservationLog]
  assignations: [Assignation!]!
}

Fields​

Reservation.id ● ID! non-null scalar​

Reservation.channel ● String! non-null scalar​

The channel of this Reservation

Reservation.happy ● Boolean! non-null scalar​

Is this reservation happy? (aka: does it have as many linked provisions as desired

Reservation.viable ● Boolean! non-null scalar​

Is this reservation viable? (aka: does it have as many linked provisions as minimal

Reservation.allowAutoRequest ● Boolean! non-null scalar​

Allow automatic requests for this reservation

Reservation.node ● Node! non-null object​

The node this reservation connects

Reservation.title ● String scalar​

A Short Hand Way to identify this reservation for you

Reservation.template ● Template object​

The template this reservation connects

Reservation.provisions ● [Provision!]! non-null object​

The Provisions this reservation connects

Reservation.params ● ReserveParams object​

Reservation.hash ● String! non-null scalar​

The hash of the Reservation

Reservation.status ● ReservationStatus! non-null enum​

Current lifecycle of Reservation

Reservation.statusmessage ● String! non-null scalar​

Clear Text status of the Provision as for now

Reservation.callback ● String scalar​

Callback

Reservation.progress ● String scalar​

Provider

Reservation.createdAt ● DateTime! non-null scalar​

Reservation.updatedAt ● DateTime! non-null scalar​

Reservation.waiter ● Waiter! non-null object​

This Reservations app

Reservation.app ● LokApp object​

This Reservations app

Reservation.creator ● User object​

This Reservations creator

Reservation.reference ● String! non-null scalar​

The Unique identifier of this Assignation

Reservation.provision ● Provision object​

Was this Reservation caused by a Provision?

Reservation.createdProvisions ● [Provision!]! non-null object​

Reservation that created this provision (if we were auto created)
